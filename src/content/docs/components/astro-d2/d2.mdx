## Styles

### Default

```d2
direction: right
Markdown definition -> D2 -> Diagram
```

### Sketch

```d2 sketch 
direction: right
Markdown definition -> D2 -> Diagram
```

## Sequences

```d2
shape: sequence_diagram

DevRel -> OSPO: What should I use to create diagrams?

```
